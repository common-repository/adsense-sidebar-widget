== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the new Widget "AdSense Anna" in your Design
4. Enter your AdSense-ID "pub-xxxxxxxxxxxxxx"
5. Choose your favorite banner
